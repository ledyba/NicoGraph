# NicoGraph
NicoGraphは、ニコニコ動画のタグを階層的にまとめあげた上で可視化するWebアプリケーションです。

超高速なグラフクラスタリングアルゴリズムを用いており、その高速性を活かして集計期間をダイナミックに変更してその場でクラスタリング・可視化をしてしまいます。

- [Demo](http://ledyba.org/NicoGraph/)
